# Angular With Lerna Experiments

## Installation

```console
yarn global add lerna@2.0.0-beta.36
```

## Bootstrap

```console
git clone https://github.com/ghoullier/angular-with-lerna.git && cd angular-with-lerna
lerna bootstrap
```

## Start

```console
cd packages/@ghoullier/ui/
ng build
````
